# Task 3: Student Feedback Sentiment Analysis

## 📌 Project Overview
This project analyzes student feedback collected after a college event.
The goal is to understand overall sentiment and identify strengths and
areas of improvement using Natural Language Processing (NLP).

## 🛠 Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- TextBlob / NLTK
- Jupyter Notebook

## 📊 Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Average rating analysis
- Top-rated survey questions
- Sentiment analysis on student feedback
- Sentiment distribution visualization

## 🔍 Key Insights
- Majority of students provided positive feedback
- Teaching quality and presentation clarity received high ratings
- Few negative responses indicate scope for improvement in some areas

## 📁 Files Included
- College_Event_Feedback_Analysis.ipynb
- student_feedback.csv
